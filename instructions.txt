Comandos de Control del Tracker (ONPE)

Dado que este es un servicio de usuario, NO debes usar sudo. Todos los comandos llevan la bandera --user.

1. Iniciar el servicio (Start)
systemctl --user start tracker.service
systemctl --user start onpe-tracker-extranjero.service
systemctl --user start onpe-tracker-peru.service

2. Detener el servicio (Stop)
systemctl --user stop onpe-tracker.service
systemctl --user stop onpe-tracker-extranjero.service
systemctl --user stop onpe-tracker-peru.service

3. Reiniciar el servicio (Restart)
systemctl --user restart onpe-tracker.service
systemctl --user restart onpe-tracker-extranjero.service
systemctl --user restart onpe-tracker-peru.service

4. Ver el estado actual (Status)
systemctl --user status onpe-tracker.service
systemctl --user status onpe-tracker-extranjero.service
systemctl --user status onpe-tracker-peru.service

5. Ver los Registros / Consola (Logs)
journalctl --user -u onpe-tracker.service -f
journalctl --user -u onpe-tracker-extranjero.service -f
journalctl --user -u onpe-tracker-peru.service -f



Para ejecutar el servicio de dashboard, se debe ejecutar el siguiente comando:
poetry run streamlit run src/elec_tracker_pe/dashboard.py
poetry run streamlit run src/elec_tracker_pe/dashboard_publico.py

Para ejecutar el servicio de affinity calculator, se debe ejecutar el siguiente comando:
poetry run python -m src.elec_tracker_pe.affinity_calculator

Para ejecutar la proyeccion, se debe ejecutar el siguiente comando:
poetry run python -m src.elec_tracker_pe.projector

Para ejecutar el servicio de auditor, se debe ejecutar el siguiente comando:
poetry run python -m src.elec_tracker_pe.auditor
